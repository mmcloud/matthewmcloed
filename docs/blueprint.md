# **App Name**: Cloud Native Pro

## Core Features:

- Homepage: Landing page highlighting Matthew McLeod's expertise in cloud-native technologies and Kubernetes.
- Services: A dedicated section detailing the specific services Matthew McLeod offers, such as cloud solutions, DevOps automation, and Kubernetes expertise.
- Blog: A blog section to share insights, articles, and updates related to cloud-native technologies and Kubernetes.
- Contact Form: A form to enable visitors to contact Matthew McLeod.
- Dashboard: A private dashboard for clients with login to show projects, their statuses, invoices, and to facilitate easy communication with Matthew.

## Style Guidelines:

- Primary color: Dark electric blue (#112D4E), conveying sophistication and confidence in cloud-native expertise.
- Background color: Slightly lighter, desaturated electric blue (#1E4577). It maintains a connection to the primary, and provides a good background for both light and dark elements.
- Accent color: Vivid sky blue (#34A7EB), providing a contrast for interactive elements and key information.
- Font: 'Inter', a versatile sans-serif for a clean, professional look. This will be used for both body and headlines.
- Glassmorphism effect for certain panels, achieved using transparency, backdrop blur, and subtle borders to create a layered, frosted-glass appearance.
- Clean, minimalist icons to represent cloud services, Kubernetes concepts, and DevOps practices.
- Subtle transitions and animations on page load and during navigation to provide a smooth, engaging user experience.
